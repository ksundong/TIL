> 해당 `StringUtils`의 의존성이 포함되어 있는지 모르겠는데
> `package org.springframework.util` 패키지의 `StringUtils`의 `isEmpty`를 활용하실수도 있을것 같네요
>
> ```java
> public static boolean isEmpty(@Nullable Object str) {
>     return (str == null || "".equals(str));
> }
> ```
