## Git checkout 없이 pull 해오는 방법

`git fetch <remote-repository> <srcBranch>:<destBranch>`

`git pull <remote> <srcBranch>:<destBranch>` 도 가능하다고 하는데 둘의 차이는 거의 없다고 볼 수 있음. (다른 브랜치가 checkout 되어있다는 가정하에)
