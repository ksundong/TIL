# Byobu 사용법

## Mac에서 Byobu 설치하는 법

```shell
brew install byobu
```

다른 OS는 알아서 찾아봅시다.

## Mac에서 Byobu 설정

```shell
byobu
(crtl+a)
```

이러면 screen 모드로 할거냐, emacs 모드로 할거냐 라고 묻는데  
나는 emacs를 전혀 쓰지 않으므로 screen mode로 지정해주었다.(emacs의 escape key와 일치한다고 함)

## Byobu 단축키 보는법

`(ctrl+a) + ?(shift+/)`

로 단축키를 볼 수 있다.

## 단점

무겁다는 얘기를 듣고 안쓰기로 마음먹었다.

tmux를 잘 써보자
